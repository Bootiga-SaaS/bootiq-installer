# Changelog
## 1.0.0 - 2026-06-19
- Initial public Bootiq storefront release.
- Removed obsolete catalog-toolbar experiments and consolidated the storefront design tokens.
- Renamed legacy Bootiga frontend classes and JavaScript behaviors to the public Bootiq namespace.
- Added responsive catalog, product detail, cart, login, legal-page, header, and footer styling.
- Added editable Hero, product grid, similar-products, and The Look Layout Builder blocks.
- Added a Composer metapackage and installer for Commerce Kickstart 5.1.
